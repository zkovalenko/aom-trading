<p>
  Consolidation patterns reveal how price pauses, compresses, or transitions between trend phases.
  Understanding these structures helps traders identify the next high-probability breakout direction.
</p>

<h2>2. Base Formation</h2>
<p>The Base represents a consolidation phase between impulsive moves and can appear as:</p>
<ul>
  <li>Doji Cluster</li>
  <li>Wedge or Fan Formation</li>
  <li>Diamond Pattern (Fan + Wedge)</li>
  <li>Hidden Consolidation (Overlapping Candles)</li>
  <li>Island Pattern (Separated by Gaps)</li>
</ul>

<h2>3. Candle Logic (R–B–D / R–B–R)</h2>
<p>Each pattern consists of three phases:</p>
<ol>
  <li><strong>R (Rally)</strong> – impulsive bullish candle</li>
  <li><strong>B (Base)</strong> – consolidation or pause</li>
  <li><strong>D (Drop)</strong> – impulsive bearish candle</li>
</ol>
<p><strong>Conviction Candle</strong> confirms breakout direction and validates the zone.</p>
